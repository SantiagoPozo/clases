Tema 7.5: Ecuaciones de Segundo Grado

Un enfoque desde la lógica algebraica

1. Recordatorio

En álgebra, solo se pueden sumar términos semejantes. Es decir, monomios que tienen exactamente la misma parte literal.

✅ \(3x^2 + 5x^2 = 8x^2\) (Son semejantes)

❌ \(x^2 + x \rightarrow\) NO se puede reducir. Son de grados distintos.

❌ \(5x^2 - 2 \rightarrow\) NO se puede reducir.

2. Ecuaciones del tipo \(x^2 = k\)

Hay \(x^2\) pero no hay \(x\). Muchas ecuaciones que no lo parecen terminan reduciéndose a este tipo.

\(x = \pm \sqrt{9}\)

Soluciones: x = 3, x = -3

3. Ecuaciones del tipo \(ax^2 + bx = 0\)

Aquí hay \(x^2\) y hay \(x\), pero no hay un número suelto (término independiente).

3.1. Primero tienes que saber esto

Una ecuación como \(x(x - 1) = 0\) se resuelve por lógica: si dos cosas multiplicadas dan cero, una de las dos tiene que ser cero.

\(x = 0\)

\(x - 1 = 0 \rightarrow x = 1\)

3.2. Sacar factor común: El viaje de ida y vuelta

Es la operación contraria a la propiedad distributiva.

Ida (Distributiva): \(x(x - 2) \rightarrow x^2 - 2x\)

Ida (Distributiva): \(5x(2x - 3) \rightarrow 10x^2 - 15x\)

Ida (Distributiva): \(3x(x + 4) \rightarrow 3x^2 + 12x\)

Ida (Distributiva): \(x(7 - x) \rightarrow 7x - x^2\)

Si vamos de derecha a izquierda, estamos sacando factor común:

\(x^2 - 2x \rightarrow x(x - 2)\)

\(10x^2 - 15x \rightarrow 5x(2x - 3)\)

\(3x^2 + 12x \rightarrow 3x(x + 4)\)

3.3. Resolviendo mediante factor común

Resuelta 1:

\(x^2 - 2x = 0\)

\(x(x - 2) = 0\)

\(x = 0\)

\(x - 2 = 0 \rightarrow x = 2\)

Soluciones: x = 0, x = 2

Resuelta 2:

\(10x^2 - 15x = 0\)

\(5x(2x - 3) = 0\)

\(5x = 0 \rightarrow x = 0\)

\(2x - 3 = 0 \rightarrow x = 1.5\)

Soluciones: x = 0, x = 1.5

4.1. Completando cuadrados

Obliga a razonar y a usar productos notables en sentido inverso.

Recuerda:

  • \((a + b)^2 = a^2 + 2ab + b^2\)
  • \((a - b)^2 = a^2 - 2ab + b^2\)
  • \((a + b)(a - b) = a^2 - b^2\)

Ejemplo: \(x^2 - 2x = 3\)

Añadimos \(+1\) para completar \((x-1)^2\).

\(x^2 - 2x + 1 = 3 + 1\)

\((x - 1)^2 = 4 \rightarrow x - 1 = \pm\sqrt{4}\)

\(x - 1 = 2 \rightarrow x = 3\)
\(x - 1 = -2 \rightarrow x = -1\)

Soluciones: x = 3, x = -1

4.2. Con la fórmula

\[x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\]

Ejemplo: \(x^2 - 5x + 6 = 0\)

  • 1. Identifica: \(a=1, b=-5, c=6\)
  • 2. Sustituye: \(x = \dfrac{5 \pm \sqrt{(-5)^2 - 4 \cdot 1 \cdot 6}}{2 \cdot 1}\)
  • 3. Opera: \(x = \dfrac{5 \pm 1}{2}\)

Soluciones: x = 3, x = 2
